# VS2017+OpenCV3.3双目立体视觉与测距资源文件

## 资源简介

本资源文件提供了基于VS2017和OpenCV3.3的双目立体视觉与测距的实现代码和相关文档。该实现主要使用了SGBM（Semi-Global Block Matching）算法进行双目校正和立体匹配，从而实现双目测距功能。文档中包含了详细的实现步骤、代码说明以及测试图片，帮助用户快速理解和应用该技术。

1. **环境配置**：
